Planejamento:

Instalar express e ejs a fim de criar as primeiras rotas
Criar uma pasta chamada views e colocar os arquivos ejs. Além disso habilitar o uso da view engine
Criar uma pasta chamada public para colocar os arquivos estáticos. Além disso habilitar o uso de arquivos estáticos. Ao instalar o bootstrap, na parte de indicar o caminho do css não colocar public na rota.
Criar uma pasta chamada partials onde serão colocados alguns arquivos ejs como  arquivo de navbar,footer e header,além do navbar administrativo.
Incluir os partials em cada pagina.

Após isso, instalar tudo o que é necessário para se conectar com o banco e efetivamente gerar uma conexão
Desenvolver as páginas administrativas


Banco:
Instalar o sequelize através do npm install --save sequelize
Instalar o mysql2 via npm install --save mysql2
Instalar o slugify
Criar um banco pro projeto
Cria uma pasta chamada database e dentro dela um arquivo database.js

Criar uma pasta chamada Articles e dentro dela colocar o model e o controller
Criar uma pasta chamada Ctaegories e dentro dela colocar o model e o controller
